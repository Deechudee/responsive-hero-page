Responsive Hero Page

This project is a responsive landing page hero section built using HTML and CSS (Flexbox).
It demonstrates how to create a modern website layout that adapts to different screen sizes such as desktop and mobile devices.

🚀 Features

Responsive Hero Section

Built using HTML5 and CSS3

Layout designed with Flexbox

Clean and simple UI

Mobile-friendly design using media queries
